## 🚀 사용 방법

### 기본 사용 (자동 선택)
```bash
auto-tessell run input.stl -o ./case
```

### 명시적 Tier 선택
```bash
auto-tessell run thin_part.stl -o ./case --tier 2d        # 2D 메시
auto-tessell run channel.stl -o ./case --tier hex         # 구조화 Hex
auto-tessell run complex.stl -o ./case --tier jigsaw_fallback  # JIGSAW
```

### 품질 레벨
```bash
auto-tessell run input.stl -o ./case --quality draft      # 빠름 (~1초)
auto-tessell run input.stl -o ./case --quality standard   # 균형 (~수분)
auto-tessell run input.stl -o ./case --quality fine       # 최고품질 (~30분+)
```

## ⚠️ 알려진 제한사항 (v0.3.1에서 개선)

### 2D 감지 개선 필요
- naca0012 같은 항공형이 2D로 인식되지 않음
- 해결: tier0_2d_meshpy의 감지 로직 강화

### Open Boundary 처리 미흡
- hemisphere_open 등 open surface 처리 실패
- 해결: L2/L3 전처리 고도화

### 극도로 복잡한 형상
- non-manifold, self-intersecting, 극도로 왜곡된 형상은 타임아웃
- 권장: Standard/Fine 품질 사용 또는 전처리 강화
