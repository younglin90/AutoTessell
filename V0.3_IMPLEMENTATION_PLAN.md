**파일**: `core/generator/tier_2d_meshpy.py` (NEW)

```python
class Tier2DMeshPyGenerator:
    """2D 메시 생성 (입구/출구/단면)."""
    
    def run(
        self,
        strategy: MeshStrategy,
        surface_path: Path,
        case_dir: Path,
    ) -> TierAttempt:
        """MeshPy를 사용한 2D 삼각형 메시 생성.

        - Triangle Delaunay 메싱
```

- 경계 조건 자동 분류 (입구/출구/벽)
- polyMesh 생성

**테스트**:
- NACA 프로필 (입구/출구)
- 덕트 단면 (경계층)
- 원형 입구

## 📋 Phase 2: 구조적 Hex 메시 (Week 1-2)

### 2.1 classy_blocks 기반 Hex Generator Tier 추가

**파일**: `core/generator/tier_hex_classy_blocks.py` (NEW)

```python
class TierHexClassyBlocksGenerator:
    """구조적 Hex 메시 (blockMeshDict Python 생성)."""
    
    def run(
        self,
        strategy: MeshStrategy,
        preprocessed_path: Path,
        case_dir: Path,
    ) -> TierAttempt:
        """classy_blocks를 사용한 구조적 Hex 메싱.
        
        1. 입력 STL 경계박스 분석
        2. blockMeshDict Python 코드 생성
        3. OpenFOAM blockMesh 실행
        4. snappyHexMesh로 표면 정렬
        """
```

**기능**:
- BBox 기반 자동 블록 분할
- 경계층 구조화 (hex 압출)
- 표면 피처 감지 및 refinement
- 최종 polyMesh 생성

**아키텍처**:
```
Input STL
    ↓
[Geometry Analysis] ← BBox, features
    ↓
[blockMeshDict Generation] ← classy_blocks
    ↓
[blockMesh] ← OpenFOAM
    ↓
[snappyHexMesh] ← 표면 정렬
    ↓
[polyMesh] ← Final
```

**테스트**:
- 간단한 상자 (기준선)
- 원기둥 (축대칭)
- 복합 형상 (표면 정렬)
