## 구현 현황

### ✅ Phase 1: 2D Mesh Generator
- **파일**: `core/generator/tier0_2d_meshpy.py` (13.0 KB)
- **클래스**: `Tier2DMeshPyGenerator`
- **기능**:
  - 3D → 2D 평면 투영 (가장 작은 분산 축 자동 감지)
  - Triangle Delaunay 메싱 (meshpy)
  - 2D Prism extrude → Tet 변환 (높이 0.001)
  - Quality level별 파라미터 조정

### ✅ Phase 2: Hex Mesh Generator (classy_blocks)
- **파일**: `core/generator/tier_hex_classy_blocks.py` (13.8 KB)
- **클래스**: `TierHexClassyBlocksGenerator`
- **기능**:
  - BBox 분석 → blockMeshDict 자동 생성
  - blockMesh 실행 (구조화 Hex 생성)
  - snappyHexMesh 실행 (표면 정렬)
  - classy_blocks 미설치 시 Fallback: 간단한 text-based blockMeshDict

### ✅ Phase 3: JIGSAW Fallback
- **파일**: `core/generator/tier_jigsaw_fallback.py` (6.1 KB)
- **클래스**: `TierJigsawFallbackGenerator`
- **기능**:
  - TetWild 실패 시 자동 전환
  - jigsawpy 기반 강건한 Tet 메싱
  - Draft/Standard 품질 레벨별 hmax/hmin 자동 조정
  - 극도로 높은 성공률 (보수적 알고리즘)

### ✅ Phase 4: 통합 테스트 및 Tier 연결
- **Tier Selector 업데이트** (`core/strategist/tier_selector.py`):
  - `_TIER_ORDER`: 9-stage fallback chain
  - `_is_2d()` 메서드: BBox 기반 2D 감지 (threshold = 0.01 * diagonal)
  - `_HINT_MAP`: "2d", "hex", "jigsaw_fallback" CLI 별칭 추가

- **Pipeline 업데이트** (`core/generator/pipeline.py`):
  - `_TIER_REGISTRY`: 3개 새로운 Tier 등록
  - `_TIER_ALIASES`: CLI 별칭 매핑

## 9-Stage Fallback Chain

```
1. tier0_2d_meshpy              (2D 메시 — 2D 기하 감지 시)
2. tier1_snappy                 (외부유동 + 경계층)
3. tier_hex_classy_blocks       (구조화 Hex)
4. tier15_cfmesh                (내부유동)
5. tier05_netgen                (CAD/일반)
6. tier0_core                   (단순형상)
7. tier_meshpy                  (Tet fallback, TetGen)
8. tier2_tetwild                (불량 표면)
9. tier_jigsaw_fallback         (최후 fallback — 매우 안정적)
```

### 자동 선택 로직 (Tier Selector)

| 입력 기하 | Quality | 선택 Tier |
|----------|---------|----------|
| 2D | Any | tier0_2d_meshpy |
| 외부유동 + Watertight | Standard | tier1_snappy |
| 내부유동 + Watertight | Standard | tier15_cfmesh |
| Watertight + Simple | Standard | tier0_core |
| 불량 표면 | Standard | tier2_tetwild |
| CAD (B-Rep) | Any | tier05_netgen |

## CLI 인터페이스

```bash
# 자동 선택 (Tier Selector 사용)
auto-tessell run input.stl -o ./case

# 명시적 Tier 선택
auto-tessell run input.stl -o ./case --tier 2d              # 2D 메시
auto-tessell run input.stl -o ./case --tier hex             # Hex 메시
auto-tessell run input.stl -o ./case --tier jigsaw_fallback # JIGSAW 강제

# 품질 레벨 선택
auto-tessell run input.stl -o ./case --quality draft        # 빠름 (TetWild)
auto-tessell run input.stl -o ./case --quality standard     # 균형 (Netgen/cfMesh)
auto-tessell run input.stl -o ./case --quality fine         # 품질 우선 (snappyHexMesh)
```
