## 📋 v0.3.1 개선 계획

### P2 우선순위
1. **naca0012 등 2D 형상 개선** (tier0_2d_meshpy 감지 강화)
2. **Open boundary 처리** (L2/L3 전처리 고도화)
3. **타임아웃 형상 최적화** (매개변수 튜닝)

### 권장사항
- Draft quality로 복잡 형상은 **Standard/Fine 사용 권장**
- Open boundary 형상은 **CAD로 closed** 또는 **domain 경계 추가** 권장
- 극도로 복잡한 형상(non-manifold 등)은 **전처리 라이브러리 사용** (MeshLab, Gmsh)
