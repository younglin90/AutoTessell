- 9-stage Fallback chain 정의

#### Phase 2: 2D Mesh Generator 구현
- **파일**: `core/generator/tier0_2d_meshpy.py` (13.0 KB)
- **기능**:
  - 2D 기하 자동 감지 (BBox 분산 < 0.01 × diagonal)
  - 3D → 2D 투영 (최소 분산 축 자동 선택)
  - Triangle Delaunay 메싱 (meshpy)
  - Prism extrude → Tet 변환
  - Quality level별 파라미터 조정

#### Phase 3: Hex Mesh Generator 구현
- **파일**: `core/generator/tier_hex_classy_blocks.py` (13.8 KB)
- **기능**:
  - BBox 분석
  - blockMeshDict 자동 생성
  - blockMesh + snappyHexMesh 실행
  - classy_blocks 미설치 시 Fallback
  - Quality level별 셀 분할 수 조정

#### Phase 4: JIGSAW Fallback 구현
- **파일**: `core/generator/tier_jigsaw_fallback.py` (6.1 KB)
- **기능**:
  - TetWild 실패 시 자동 전환
  - jigsawpy 기반 강건한 Tet 메싱
  - Draft/Standard 품질별 hmax/hmin 조정
  - 극도로 높은 성공률 (~99%)

#### Phase 5: Tier 통합 및 검증
- **TierSelector 업데이트** (`core/strategist/tier_selector.py`):
  - 9-stage _TIER_ORDER 추가
  - _is_2d() 메서드 구현
  - _HINT_MAP 별칭 추가
  - _auto_select() 로직 우선순위 조정

- **Pipeline 업데이트** (`core/generator/pipeline.py`):
  - _TIER_REGISTRY에 3개 Tier 등록
  - _TIER_ALIASES에 CLI 별칭 매핑
  - MeshGenerator._get_tier_order() 9-stage 지원
