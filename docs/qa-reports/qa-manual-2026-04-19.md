# AutoTessell Manual GUI QA

**작성일:** 2026-04-19  
**기준 리포트:** `.gstack/qa-reports/qa-manual-2026-04-19.md`  
**상태:** headless 환경에서 자동 개선 착수, 실기계 QA는 미수행

## 이번 개선에서 처리한 항목

1. QA 문서가 참조하는 실행 명령 `python3 -m desktop.qt_app`이 동작하도록
   `desktop/qt_app/__main__.py`를 추가했다.
2. 실제 디스플레이가 있는 Windows/WSL X11/Wayland 환경에서 PyVistaQt가 네이티브
   OpenGL 렌더링을 사용할 수 있도록 `desktop/qt_main.py`의 PyVista runtime 설정을
   정리했다.
   - headless/offscreen일 때만 `pyvista.OFF_SCREEN=True`
   - Windows 또는 `DISPLAY`/`WAYLAND_DISPLAY`가 있는 Linux/WSL에서는 offscreen 강제 안 함

## Headless 검증 기준

```bash
python3 -m pytest tests/test_qt_app.py tests/test_gui_visual.py -q
```

이 리포트 작성 시점의 이전 기준은 `185 passed, 8 skipped`였다.
